# 물결표(~) 취소선 오작동 방지 규칙

## 배경
채팅은 Markdown으로 렌더링됩니다. 한 문장(또는 한 줄) 안에 `~`가 2개 있으면
`~내용~`이 **취소선(strikethrough)**으로 해석되어, 의도하지 않은 취소선이 표시됩니다.
(예: `약 8~12턴`, `10~15턴`, `1~3개` 처럼 범위를 나타내려던 `~`가 취소선이 됨)

## 규칙
- 취소선을 **의도하지 않은** 경우, 한 문장/한 줄 안에서 `~`가 2개 이상 나오지 않도록 처리합니다.
- 다음 방법 중 하나로 `~`가 원문 그대로(취소선 없이) 보이도록 작성합니다.

### 권장 처리 방법 (우선순위 순)
1. **범위 표기는 물결표 대신 하이픈/문구 사용**
   - `8~12턴` → `8-12턴` 또는 `8에서 12턴`
2. **역슬래시로 이스케이프**
   - `약 8\~12턴` 처럼 각 `~` 앞에 `\`를 붙입니다.
3. **인라인 코드로 감싸기**
   - `` `8~12턴` `` 처럼 백틱으로 감싸면 `~`가 그대로 표시됩니다.

## 예외
- **실제로 취소선을 의도한 경우**에는 이 규칙을 적용하지 않고 `~취소할내용~`을 그대로 사용합니다.
- 코드 블록(```` ``` ````) 안의 `~`는 렌더링되지 않으므로 그대로 두어도 됩니다.
